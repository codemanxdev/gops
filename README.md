## Build Process

#### Install Dependences

- npm install

#### Compile

- npm run compile

#### Build

- npm run build

#### Check the files that will be packaged

- vsce ls

#### Package

- vsce package --no-dependencies

#### Publish

- vsce publish --no-dependencies
